# cowsay.sh

Simple clone of `cowsay` written in POSIX compliant shell.

## Usage

```sh
./cowsay.sh hello
```

prints

```
 .─────. 
| hello |
 '─────' 
        \   ^__^
         \  (OO)\_______
            (__)\       )\/\
                ||----w |
                ||     ||
```

Longer text would be split to multiline.

## Installation

Put `cowsay.sh` to some directory existing in your `$PATH`.
